Selection.removeAllRanges()

The Selection.removeAllRanges() method removes all ranges from the selection, leaving the anchorNode and focusNode properties equal to null and leaving nothing selected.

Syntax

removeAllRanges()

Parameters

None.

Return value

None (undefined).
