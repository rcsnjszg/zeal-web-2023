SecurityPolicyViolationEvent.originalPolicy

The originalPolicy read-only property of the SecurityPolicyViolationEvent interface is a string containing the policy whose enforcement uncovered the violation.

Value

A string representing the policy whose enforcement uncovered the violation.

Examples

document.addEventListener("securitypolicyviolation", (e) => {
  console.log(e.originalPolicy);
});
